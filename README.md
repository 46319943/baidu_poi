# 使用百度地图提供的地点检索API，爬取各个城市的POI点
- 指定行政区名称以及搜索关键词即可
- 异步请求，爬取速度大概是一千条每秒
- 关键参数：
    - scrape_bound函数中的total_threshold，决定了POI查找是否完全
    - if f'{min_x:.4f}' != f'{max_x:.4f}':
        - 避免分割多次之后总数仍超过阈值而导致无限分割
        - 当分割小到一定程度，根据边界坐标精度停止
